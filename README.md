# Dashboard-Data-Sales

## Deskripsi Proyek

Proyek ini bertujuan untuk menganalisis performa penjualan retail menggunakan Tableau Public melalui visualisasi data interaktif yang berfokus pada metrik utama seperti total penjualan, profit, dan biaya pengiriman. Dashboard ini dirancang untuk membantu pengguna memahami performa bisnis berdasarkan wilayah dan kategori produk sehingga dapat mendukung pengambilan keputusan yang lebih efektif.

## Tools yang Digunakan

* Tableau Public
* Microsoft Excel (Data Source)
* Data Cleaning dan Data Preparation

## Fitur Dashboard

* Visualisasi total Sales Year-to-Date (YTD)
* Visualisasi total Profit Year-to-Date (YTD)
* Analisis Sales berdasarkan Region
* Analisis Profit berdasarkan Region
* Analisis Average Shipping Cost berdasarkan Region
* Filter interaktif berdasarkan Region dan Product Category

## Insight Utama

* Region Central memiliki total penjualan tertinggi sebesar $2.54M.
* Region Central juga menghasilkan profit tertinggi sebesar $519K.
* Region South memiliki performa penjualan dan profit terendah dibandingkan region lainnya.
* Biaya pengiriman rata-rata tertinggi terdapat pada region East.
* Dashboard memungkinkan pengguna melakukan eksplorasi data lebih lanjut berdasarkan kategori produk dan wilayah tertentu.

## Tujuan Bisnis

Membantu stakeholder dalam:

* Mengidentifikasi wilayah dengan performa terbaik dan terendah.
* Memantau profitabilitas setiap region.
* Mengevaluasi efisiensi biaya pengiriman.
* Mendukung strategi penjualan dan distribusi yang lebih efektif.
Tahapan Pembuatan Dashboard Tableau
1. Data Preparation
Mengimpor dataset penjualan retail ke Tableau.
Memastikan tipe data sudah sesuai seperti Sales, Profit, Shipping Cost, Region, dan Product Category.
Melakukan pengecekan data kosong dan duplikasi data.
2. Membuat Dashboard Sales

Membuat dua visualisasi utama:

KPI Total Sales YTD sebesar $8,951,870.6.
Bar chart Sales berdasarkan Region:
Central : $2,540,280.9
East : $2,422,804.7
West : $2,391,438.8
South : $1,597,346.2
3. Membuat Dashboard Profit

Membuat visualisasi profit:

KPI Profit YTD sebesar $1,312,442.4.
Bar chart Profit berdasarkan Region:
Central : $519,825.6
East : $377,566.2
West : $310,849.5
South : $104,201.2
4. Membuat Dashboard Sales and Profit

Menggabungkan metrik Sales dan Profit untuk melihat hubungan antara penjualan dan keuntungan pada masing-masing region.

5. Membuat Dashboard Average Shipping Cost

Menganalisis biaya pengiriman rata-rata:

Average Shipping Cost YTD sebesar $2.1.
Region East memiliki rata-rata biaya pengiriman tertinggi.
Region South memiliki biaya pengiriman rata-rata terendah.
6. Menambahkan Filter Interaktif

Menambahkan filter:

Region
Product Category

Filter diterapkan ke seluruh dashboard sehingga pengguna dapat melakukan analisis yang lebih spesifik.

7. Membuat Story Dashboard

Menggunakan fitur Story pada Tableau untuk menyusun alur analisis:

Overview Sales Dashboard
Profit Dashboard
Sales and Profit Dashboard
Shipping Cost Dashboard
Dashboard Summary
